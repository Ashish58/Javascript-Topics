Running Code in Dev Tools (Line by Line)

Description:

JavaScript code can be debugged using browser developer tools. You can step through the code line by line to inspect variable values and understand code execution.

Example:

To debug JavaScript code in your browser's developer tools:

  • Open DevTools (usually by pressing F12 or Ctrl+Shift+I / Cmd+Option+I).
  • Go to the "Sources" tab.
  • Set a breakpoint by clicking on the line number where you want to pause execution.
  • Use the "Step Over" (F10) and "Step Into" (F11) buttons to execute code line by line and inspect variable values.

Practice Question:

Set a breakpoint in the following function and observe variable changes line by line. 

function add(a, b) {
    let result = a + b;
    console.log(result);
    return result;
}
add(5, 10);
