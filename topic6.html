Template Literals, String Methods & Properties

Description:

Template literals use backticks ( `` ) to create strings that can include expressions and multiline text. They enhance string manipulation capabilities.

Example:

let name = "John";
let greeting = `Hello, ${name}!`;
let multiline = `This is line 1
This is line 2`;

Practice Question:

Use template literals to construct a message that includes a variable name and a multiline address.

let userName = "Alice";
let userAddress = `Name: ${userName}
Address: 1234 Elm St.
City: Springfield`;
console.log(userAddress);

Output Question:

What is the output of this program?

let userName = "Alice";
let userAddress = `Name: ${userName}
Address: 1234 Elm St.
City: Springfield`;
console.log(userAddress);

Expected Output:
