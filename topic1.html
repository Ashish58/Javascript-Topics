Description:

JavaScript is a high-level programming language used to create interactive and dynamic content on websites. It runs in the browser, allowing developers to manipulate the DOM (Document Object Model) and enhance user experiences.

Example:

HTML:

<h1 id="title">Welcome!</h1>

JavaScript:

document.getElementById('title').textContent = 'Welcome to JavaScript!';

Explanation: This script changes the text inside the <h1> tag from "Welcome!" to "Welcome to JavaScript!" upon page load.

Practice Question:

Create an HTML button that, when clicked, changes the text of a paragraph to "Button Clicked!".

Expected Output: The paragraph text should change to "Button Clicked!" upon clicking the button. 

<button onclick="document.getElementById('message').textContent = 'Button Clicked!'>Click Me</button>
<p id="message">Original Text</p>

Output Question:

<button onclick="document.getElementById('message').textContent = 'Button Clicked!'">Click Me</button>
<p id="message">Original Text</p>

What is the output of this program when the button is clicked?

Expected Output: The paragraph text will change to "Button Clicked!" when the button is clicked.
