Truthy and Falsy Values in JavaScript

Description:

In JavaScript, values are either "truthy" or "falsy" in conditional contexts. Falsy values include false, 0, "" (empty string), null, undefined, and NaN. All other values are considered truthy.

Example:

let value = 0;
if (value) {
    console.log("Value is truthy");
} else {
    console.log("Value is falsy");
}

Practice Question:

Write a function that checks if a variable is truthy or falsy and returns a corresponding message.

function checkValue(val) {
    return val ? "Value is truthy" : "Value is falsy";
}
console.log(checkValue("Hello")); // What is the output?
console.log(checkValue(0));       // What is the output?
