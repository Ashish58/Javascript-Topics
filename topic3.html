JavaScript Variables (let, const, var)

Back to Topics

Description:

JavaScript variables can be declared using:

  • let: Block-scoped, mutable.
  • const: Block-scoped, immutable; must be initialized at declaration.
  • var: Function-scoped; avoid using var in modern code due to potential issues with scope.

Example:

// Variables Declaration
let count = 10;       // Block-scoped, mutable
const PI = 3.14159;   // Block-scoped, immutable
var globalVar = "I'm a global variable"; // Function-scoped

Practice Question:

Write a function using let and const inside a block scope and explain the difference in behavior compared to var.

Example:

function testScopes() {
    if (true) {
        var x = 1;   // Function-scoped
        let y = 2;   // Block-scoped
        const z = 3; // Block-scoped
    }
    console.log(x); // What is the output?
    console.log(y); // What is the output?
    console.log(z); // What is the output?
}
testScopes();
