Dialog Boxes in JavaScript (Alert, Confirm, Prompt)

Description:

JavaScript provides built-in dialog boxes for user interaction:

  • alert(): Displays a message.
  • confirm(): Asks for user confirmation (OK/Cancel).
  • prompt(): Requests user input.

Example:

alert("Welcome to the site!");
let userResponse = confirm("Do you agree?");
let userName = prompt("Enter your name:", "Guest");

Practice Question:

Create a script that asks the user for their age using prompt(), then displays an alert based on whether the age is above or below 18. 

let age = prompt("Enter your age:");
if (age >= 18) {
    alert("You are an adult.");
} else {
    alert("You are a minor.");
}
