Math Object in JavaScript

Description:

The Math object provides properties and methods for performing mathematical operations.

Example:

let randomNum = Math.random(); // Returns a random number between 0 and 1
let roundedNum = Math.floor(4.7); // Returns 4

Practice Question:

Write a function that generates a random integer between two given values using the Math object.

function getRandomInt(min, max) {
    return Math.floor(Math.random() * (max - min + 1)) + min;
}
console.log(getRandomInt(1, 10));

Output Question:

What is the output of this program?

function getRandomInt(min, max) {
    return Math.floor(Math.random() * (max - min + 1)) + min;
}
console.log(getRandomInt(1, 10));

Expected Output:

A random integer between 1 and 10 (inclusive).
